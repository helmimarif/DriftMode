# DriftMode

### What is this?
- This is a resource which adds drift mode

### Features
- To drift press **`LSHIFT`**
- Optimized

## Download & Installation

### Using Git
```
git clone https://github.com/helmimarif/DriftMode
```

### Manually
- Download from this

## Installation
- Add this to your `server.cfg`

```
start DriftMode
```
